About

Space bounce is an exciting new arcade adventure, set in space. The game creates a dynamic yet simple experiene for players who just want to pick up and play, complete with and atmospheric setting and story.

But more importantly this project was created to demonstrate the power and ease HTML5 and javascript brings to aspiring web developers and designers. Developed with ease using the Box2D physics engine and the feauture rich Createjs framework (both open source and well supported), Space Bounce shows developers and designers have everything they need to create slick interactive experiences without reliance on propretary Adobe products such as Flash.
